## 🚀 Getting Started

```bash
# 📦 Install dependencies
npm install

# 💻 Run in development mode
npm run dev

# 🏗️ Build for production (outputs to /docs)
npm run build
```

---

## 🌐 Deployment

The production build outputs to the `docs/` directory, making it ready for **GitHub Pages** deployment out of the box. Just push and enable GitHub Pages pointing to `/docs` on the `main` branch! 🎉
